Como administrador, puedes:

Iniciar sesión en el portal con tu nombre de usuario y contraseña para gestionar la plataforma de manera segura
Cerrar sesión en el portal para proteger el acceso al sistema
Agregar doctores al portal
Eliminar el perfil de un doctor del portal
Ejecutar un procedimiento almacenado en MySQL CLI para obtener el número de citas por mes y rastrear las estadísticas de uso

Como paciente, puedes:

Ver una lista de doctores sin iniciar sesión para explorar opciones antes de registrarte
Registrarte usando tu correo electrónico y contraseña para reservar citas
Iniciar sesión en el portal para gestionar tus reservas
Cerrar sesión en el portal para asegurar tu cuenta
Iniciar sesión y reservar una cita de una hora para consultar con un doctor
Ver mis próximas citas para poder prepararme adecuadamente

Como doctor, puedes:

Iniciar sesión en el portal para gestionar tus citas
Cerrar sesión en el portal para proteger mis datos
Ver mi calendario de citas para mantenerme organizado
Marcar tu indisponibilidad para informar a los pacientes solo sobre los horarios disponibles
Actualizar tu perfil con especialización e información de contacto para que los pacientes tengan información actualizada
Ver los detalles del paciente para las citas próximas para poder estar preparado
